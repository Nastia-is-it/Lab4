## 6. Определить, попадает ли заданное число в отрезок. Унарные операции ! - длина отрезка, ++ - увеличить координаты границ отрезка на 1. Операции приведения типа: int (явная) – целая часть координаты х; double (неявная) – результатом является координата y. Бинарные операции: +int d - целое число (координаты границ увеличиваются на число d), < LineSegment, целое число – результат равен true, если целое число попадает в заданный отрезок, иначе false.

* Метод InTheSegment(double value) проверяет принадлежность точки к отрезку.
* Start(): Возвращает начальную точку отрезка
* End(): Возвращает конечную точку отрезка

Перегруженные операторы

Унарные операторы:
* !segment - длина отрезка
* ++segment и segment++ - сдвиг отрезка на 1 вправо

Операторы преобразования: 
* Явное преобразование в int - возвращает целую часть начальной точки
* Неявное преобразование в double - возвращает конечную точку

Бинарные операторы:
* segment + d и d + segment - Сдвигает отрезок на d единиц
* segment < value - Проверяет, принадлежит ли целое число отрезку
* segment > value - Проверяет, не принадлежит ли целое число отрезку
